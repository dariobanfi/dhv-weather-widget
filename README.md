# DHV Weather Widget

An Android home screen widget that fetches and displays paragliding weather forecasts from [dhv.de](https://www.dhv.de/wetter/dhv-wetter/). It parses the forecast for Germany, Northern Alps, and Southern Alps and displays them in a clean, Material 3 style widget.

> **Note:** This is coded entirely with Gemini CLI without one single line of code changed by hand.
